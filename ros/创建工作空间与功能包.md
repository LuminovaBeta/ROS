# 创建工作空间
1、
```
mkdir catkin_ws
```
2、编译
```
cd ~/catkin_sw/
catkin_make
```
# 创建功能包
1、创建
```
cd ~/ctakin_ws/src/
catkin_create_pkg test_pkg roscpp rospy std_msgs
```
2、编译
```
cd ~/catkin_ws/
catkin_make
```
3、设置环境变量
```
source ~/catkin_ws/devel/setup.bash
```
避免每次都要执行这个脚本文件，可以把这个文件拷贝到主目录的 **.bashrc** (隐藏文件)这个文件里的最后
```
source /home/[用户名]/catkin_ws/devel/setup.bash
```
4、检查环境变量
```
echo $ROS_PACKAGE_PATH
```
