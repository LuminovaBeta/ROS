# 创建工作空间
1、
```
mkdir catkin_ws
```
2、编译
```
cd ~/catkin_sw/
catkin_make
```
# 创建功能包
1、创建
```
cd ~/ctakin_ws/src/
catkin_create_pkg test_pkg roscpp rospy std_msgs
```
2、编译
```
cd ~/catkin_ws/
catkin_make
```
设置环境变量
```
source ~/catkin_ws/devel/setup.bash
```
检查环境变量
```
echo $ROS_PACKAGE_PATH
```
